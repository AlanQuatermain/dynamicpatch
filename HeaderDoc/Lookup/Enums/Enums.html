Platform Architecture Constants

enum { 
    kInsertionArchPPC = 0, 
    kInsertionArchIA32 = 1  
};
Constants
kInsertionArchPPC
PowerPC architecture
kInsertionArchIA32
IA-32 (i386+) architecture
Discussion

These constants are mostly used when injecting PPC code into a Rosetta process. However, those functions also require cross-architecture function address lookups for finding PPC addresses from Intel code. As such, the constans are defined here and are passed to FindFunctionForArchitecture to enable those lookups.
